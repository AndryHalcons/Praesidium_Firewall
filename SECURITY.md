# Security policy

PraesidiumFirewall is beta software. Do not expose the management interface to untrusted networks.

## Reporting security issues

Report security issues privately to the repository maintainer. Include reproduction steps, affected files/endpoints, expected impact, and any logs needed to verify the issue.

## Operational guidance

- Use HTTPS and strong credentials.
- Restrict management access by network.
- Review sudoers entries before deployment.
- Replace development certificates and keys.
- Run the safe and security test profiles before release.
